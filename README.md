# Halo AAC

Augmentative and alternative communication (AAC) web app. Tap picture cards to build sentences, then speak them aloud.

## Features

- **Word packs** — Essential, School, Home, or Everything
- **Multiple profiles** — Each with own custom words and settings
- **Custom words** — Look up any word, pick an emoji, add it
- **Parent Portal** — Password-protected panel for managing words
- **Cloud sync** — Sign in with Google to sync data across devices
- **System TTS** — Uses the browser's built-in speech synthesis
- **Theming** — Custom accent colors and background
- **Offline-friendly** — Core functionality works without internet

## Usage

Open `index.html` in any modern browser. No build step needed.

Tap a category, then a word card to add it to the message bar. Hit **Speak** to hear it aloud.
